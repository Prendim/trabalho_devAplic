SPAN

Tem como objetivo agrupar textos ou tags no documento
(semelhante à DIV), utilizada pra aplicar CSS normalmente
para os textos sem apresentar alterações semânticas
(como negrito, itálico por exemplo).
Essa tag pode ter todos os atributos globais (ID, class, style...)
e no CSS não tem nenhum atributo padrão definido, ou seja,
naturalmente não oferece nenhuma alteração visual.
