GRID/TEMPLATE-COLUMNS

A propriedade css GRID oferece um sistema de layout
baseado em grid, com linhas e colunas, facilitando o design
de páginas web sem a necessidade de usar floats e posicionamento.
Já a propriedade GRID-TEMPLATE-COLUMNS é utilizada para
especificar o número de colunas dentro do grid.
